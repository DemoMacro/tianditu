## Getting started

### CDN

```html
<html>
  <head>
    <script
      type="module"
      src="https://cdn.jsdelivr.net/npm/@tianditu/wc/dist/web-components/web-components.esm.js"
    ></script>
  </head>
  <body>
    <tdt-map tk=""></tdt-map>
  </body>
</html>
```

### NPM

```bash
# npm
$ npm install -D @tianditu/wc

# yarn
$ yarn add -D @tianditu/wc

# pnpm
$ pnpm add -D @tianditu/wc
```
