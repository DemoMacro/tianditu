## Getting started

```bash
# npm
$ npm install -D @tianditu/web-components @tianditu/vue

# yarn
$ yarn add -D @tianditu/web-components @tianditu/vue

# pnpm
$ pnpm add -D @tianditu/web-components @tianditu/vue
```
